# Layout-opties

Generieke set lay-outrichtingen voor een concept-pagina, elk met een kort plus- en minpunt. Niet projectspecifiek — herbruikbaar voor elk paginaconcept.

1. **Klassiek boven-onder** — Grote titel/pitch bovenaan, secties eronder in vaste volgorde, contact als laatste blok.
   + Overzichtelijk, voorspelbaar voor bezoekers.
   – Kan saai ogen zonder sterk beeldmateriaal.

2. **Uitgelicht cijfer/quote** — Bovenaan één sterk cijfer, statistiek of quote als blikvanger, rest eronder.
   + Trekt meteen aandacht, goed voor een impact-boodschap.
   – Werkt alleen als er een sterk cijfer of quote beschikbaar is.

3. **Eén-pagina scroll** — Alles op één lange pagina, met interne links/anker-navigatie.
   + Simpel te bouwen en te delen (één link).
   – Wordt lang bij veel content.

4. **Losse pagina's** — Elk onderdeel (bijv. hero, contact) een eigen pagina met menu.
   + Schaalt beter bij veel content.
   – Meer bestanden, meer navigatie om te bouwen binnen de appetite.
